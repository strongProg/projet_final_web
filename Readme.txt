Nom :
Acheter un billet 
cette page web permet de saisir des données pour l'achat d'un billet d'avion

date: Octobre 2022
Programmeur: Armstrong Beauzil

Description:
ce programme contient deux pages, une page de saisie et une page d'affichage d'un modèle de billet à l'aide du javascript.
cete page a été réalisé avec une page html dans laqu'elle une balise formulaire est déclarée avec so tag reference name= "enregistré
des champs inputs ont été inséré dans cet objet dit formulaire. Du javascript a été utilisé de sorte que si l'utisateur
rentre une valeur vide dans l'une des champs, le programme reagi en précisant que tous les champs doivent avoir une valeur.

La deuxième page ne fait que afficher un modèle d'un billet quelquonque à l'aide du javascript.

Installation
Aucun module externe n'a été utilisé dans la réalisation du page web.

Utilisation
A l'execution du programme trois choix sont offerts, une page d'acceui pour la description dde la page web.
une page de saisie de données, une page d'affichage d'un specimen de ticket.

  Ce programme n'est pas fonctionnelle, elle a été réalisée dans le cadre d'un projet d'examen, nous sommes encore en phase d'implementation. d'autres améliorations sont à venir.
